# Superstore-analysis
This repository contains the Superstore dataset used for data analysis and visualization.  The dataset includes sales, profit, customer, and product-related information. It is useful for practicing data cleaning, exploratory data analysis (EDA), and building dashboards using tools like Python, SQL, and Power BI.
